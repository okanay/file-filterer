## Node JS

You can download Node JS using this link.

```bash
https://nodejs.org/dist/v20.10.0/node-v20.10.0.pkg
```

## Getting Started

First, install all the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Finally, open the project:

```bash
http://localhost:3000/
```
